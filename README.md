# Single-Page CV

A single-page CV built using only HTML.

## Project URL

https://roadmap.sh/projects/single-page-cv

## Features

- Semantic HTML
- SEO Meta Tags
- Open Graph Tags
- Favicon
- Single-page layout

## How to Run

1. Clone the repository

```bash
git clone https://github.com/AnoopGeorge418/SinglePage-CV.git
```

2. Open `index.html` in your browser.
